自建云盘？试试 Cloudreve

  • 作者: 傅怡然
  • 时间: 2023-08-05
  • 分类: 科技

介绍：

Cloudreve 一款自用公用都合适的网盘系统，支持多种云存储平台，例如七牛，OSS，COS，又拍云，OneDrive，S3。
用户在实际使用时无须关心物理存储方式。既可以使用 Cloudreve 搭建起个人用网盘、文件分享系统，也可以搭建针对大小团体的公有云系统。

部署方法一：使用Docker安装

第一步：更新源

apt update -y && apt upgrade -y        #Ubuntu/Debian
yum update -y && yum upgrade -y        #Centos

第二步：安装并配置Docker

安装Docker：

curl -fsSL https://get.docker.com -o get-docker.sh && sudo sh ./get-docker.sh

启动Docker：

sudo systemctl start docker

设置Docker在开机时自动启动：

sudo systemctl enable docker

第三步：安装Aria2（如果不需要离线下载可以直接跳到第四步）

记得对<TOKEN>进行修改

docker run -d \
    --name aria2-pro \
    --restart unless-stopped \
    --log-opt max-size=1m \
    -e PUID=$UID \
    -e PGID=$GID \
    -e UMASK_SET=022 \
    -e RPC_SECRET=<TOKEN> \
    -e RPC_PORT=6800 \
    -p 6800:6800 \
    -e LISTEN_PORT=6888 \
    -p 6888:6888 \
    -p 6888:6888/udp \
    -v $PWD/aria2-config:/config \
    -v $PWD/aria2-downloads:/downloads \
    p3terx/aria2-pro

第四步：创建目录结构

mkdir -vp cloudreve/{uploads,avatar} \
&& touch cloudreve/conf.ini \
&& touch cloudreve/cloudreve.db

第五步：安装并启动 Cloudreve 容器

docker run -d \
-p 5212:5212 \
--mount type=bind,source=<path_to_your_config>,target=/cloudreve/conf.ini \
--mount type=bind,source=<path_to_your_db>,target=/cloudreve/cloudreve.db \
-v <path_to_your_uploads>:/cloudreve/uploads \
-v <path_to_your_avatar>:/cloudreve/avatar \
cloudreve/cloudreve:latest

运行完成后可以访问 http://<your-ip>:5212 来访问Cloudreve

部署方法二：从打包完成的主程序进行构建

第一步：更新源并安装依赖

Ubuntu/Debian：

apt update -y && apt upgrade -y

Centos：

yum update -y && yum upgrade -y

第二步：安装并配置Docker

安装Docker：

curl -fsSL https://get.docker.com -o get-docker.sh && sudo sh ./get-docker.sh

启动Docker：

sudo systemctl start docker

设置Docker在开机时自动启动：

sudo systemctl enable docker

第三步：安装Aria2（如果不需要离线下载可以直接跳到第四步）

记得对<TOKEN>进行修改

docker run -d \
    --name aria2-pro \
    --restart unless-stopped \
    --log-opt max-size=1m \
    -e PUID=$UID \
    -e PGID=$GID \
    -e UMASK_SET=022 \
    -e RPC_SECRET=<TOKEN> \
    -e RPC_PORT=6800 \
    -p 6800:6800 \
    -e LISTEN_PORT=6888 \
    -p 6888:6888 \
    -p 6888:6888/udp \
    -v $PWD/aria2-config:/config \
    -v $PWD/aria2-downloads:/downloads \
    p3terx/aria2-pro

第四步：获取 Cloudreve

在 GitHub Release 页面获取已经构建打包完成的主程序，选择适合系统架构下可用的主程序，命名规则为cloudreve_版本号_操作系统_CPU架构.tar.gz 。

第五步：解压并执行主程序

tar -zxvf cloudreve_VERSION_OS_ARCH.tar.gz
chmod +x ./cloudreve
./cloudreve

第六步：部署进程守护

  1. 编辑配置文件
    vim /usr/lib/systemd/system/cloudreve.service

  2. 将下文 PATH_TO_CLOUDREVE 更换为程序所在目录：

    [Unit]
Description=Cloudreve
Documentation=https://docs.cloudreve.org
After=network.target
After=mysqld.service
Wants=network.target

[Service]
WorkingDirectory=/PATH_TO_CLOUDREVE
ExecStart=/PATH_TO_CLOUDREVE/cloudreve
Restart=on-abnormal
RestartSec=5s
KillMode=mixed

StandardOutput=null
StandardError=syslog

[Install]
WantedBy=multi-user.target

  3. 管理指令：

    # 更新配置
systemctl daemon-reload

# 启动服务
systemctl start cloudreve

# 设置开机启动
systemctl enable cloudreve

# 停止服务
systemctl stop cloudreve

# 重启服务
systemctl restart cloudreve

# 查看状态
systemctl status cloudreve

    运行完成后可以访问 http://<your-ip>:5212 来访问Cloudreve
