如何搭建Zbook（自己的电子书下载网站）

  • 作者: 傅怡然
  • 时间: 2023-08-01
  • 分类: 科技

介绍：

Zbook，一个电子书下载网站，官方提供了可直接下载安装的电子书文件包，大大简化部署流程，数据查找迅速，界面美观。

安装流程：

第一步：更新源

#Ubuntu/Debian
apt update -y && apt upgrade -y        
#Centos
yum update -y && yum upgrade -y

第二步：配置Docker

  1. 安装Docker
curl -fsSL https://get.docker.com -o get-docker.sh && sh ./get-docker.sh
  1. 启动 Docker
sudo systemctl start docker
  1. 设置 Docker 在开机时自动启动
sudo systemctl enable docker

第三步：下载 zbook.tar.gz 文件：

wget https://index.zbook.lol/zbook.tar.gz

第四步：解压 zbook.tar.gz 文件：

tar -xf zbook.tar.gz -C /path/to/your/folder

替换 /path/to/your/folder 为实际想要解压到的文件夹路径。

第五步：删除 zbook.tar.gz 文件以释放储存空间

rm zbook.tar.gz

第六步：运行 Docker：

docker run -d -p 8080:80 -v /path/to/your/folder/index:/index zbook/zbook:latest

替换 /path/to/your/folder 为你在第四步中解压文件的实际文件夹路径。

配置完成，可以访问网站啦，访问"IP：8080"即可

上传自己的书籍：

上传书籍至 https://libgen.is ，官方会在收录书籍后进行更新
