如何部署 Uptime Kuma — 一款进行多方面监测的工具，不仅限于网站哦

  • 作者: 傅怡然
  • 时间: 2023-08-03
  • 分类: 科技

介绍：

Uptime Kuma，一款能让你更安心的监测工具，可监测网站，数据库，Docker容器，甚至能监测Steam游戏服务器，集成了近50种通知方式用于通知实时情况，可直接配置代理和反向代理无需费心安装，使用更轻松。

效果图：


缺点：

适合自用，可以分享界面来进行无需登录的监测查看，但无法配置用户系统，只能使用一个管理员账户来进行管理。

部署方法一：使用Docker安装

第一步：更新源

Ubuntu/Debian：

apt update -y && apt upgrade -y

Centos：

yum update -y && yum upgrade -y

第二步：安装Docker

curl -fsSL https://get.docker.com -o get-docker.sh && sh ./get-docker.sh

第三步：进行部署

docker run -d --restart=always -p 3001:3001 -v uptime-kuma:/app/data --name uptime-kuma louislam/uptime-kuma:1

部署完成后可以访问 http://<your-ip>:8090 来访问这个Web界面

部署方法二：从源代码安装

第一步：更新源并安装依赖

Ubuntu/Debian：

apt update -y && apt upgrade -y 
sudo apt install git -y
sudo apt install curl -y

Centos：

yum update -y && yum upgrade -y   
sudo yum install git -y
sudo yum install curl -y

第二步：安装Node.js

Uptime Kuma需要Node.js 14、16、18或20.4。在这里，我选择的是安装Node.js 16来作为示例。

运行以下命令来安装Node.js 16：

Ubuntu/Debian：

curl -sL https://deb.nodesource.com/setup_16.x | sudo -E bash -
sudo apt-get install -y nodejs

Centos：

curl -sL https://rpm.nodesource.com/setup_16.x | sudo -E bash -
sudo yum install -y nodejs

验证Node.js和npm的安装：

node -v
npm -v

第三步：克隆仓库

git clone https://github.com/louislam/uptime-kuma.git

第四步：设置Uptime Kuma

cd uptime-kuma
npm run setup

第五步：安装和配置PM2

sudo npm install pm2 -g
pm2 install pm2-logrotate

第六步：使用PM2启动Uptime Kuma

pm2 start server/server.js --name uptime-kuma

部署完成后可以访问 http://<your-ip>:3001 来访问这个Web界面

第七步：配置PM2自启动

如果希望Uptime Kuma在系统启动时自动运行，执行以下命令：

pm2 save
pm2 startup
